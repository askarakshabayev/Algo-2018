Grasshopper: x+1, 2x (N = 15 example) 
For example, suppose that N = 15. Then greedy
algorithm will give the following 11 steps:
0 → 1 → 2 → 4 → 8 → 9 → 10 → 11 → 12 → 13 → 14 → 15.
But optimal solution in this case is 7 steps:
0 → 1 → 2 → 3 → 6 → 7 → 14 → 15.
